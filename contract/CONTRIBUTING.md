# Contributing to Escrow Smart Contract

Thank you for your interest in contributing!

## How to Contribute

1. **Fork and Clone the Repository

```bash
git clone <your-fork-url>
cd contract
```

2. **Install Dependencies

Make sure you have Rust and `cargo-soroban` installed.

3. **Create a Branch**

```bash
git checkout -b feature/your-feature
```

4. **Make Your Changes**

- Write clean, readable code
- Add tests for your changes
- Run tests to ensure everything works

5. **Run the Tests**

```bash
cargo test
```

6. **Commit Your Changes**

Follow conventional commits:
```
feat(scope): brief description
```

7. **Open a Pull Request**

## Code Style

- Follow Rust standard formatting: run `cargo fmt` before committing
- Use `cargo clippy` to catch lint issues

## Issues

Feel free to open an issue if you find a bug or have an idea for improvement!
